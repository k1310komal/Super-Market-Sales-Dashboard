# Supermarket Sales Dashboard

## Project Overview

This project presents an interactive Power BI dashboard built to analyze supermarket sales data and uncover valuable business insights. The dashboard helps visualize sales performance, customer purchasing patterns, product category trends, and overall business growth through dynamic charts and KPIs.

## Objectives

* Analyze overall sales performance.
* Identify top-performing product categories.
* Track revenue and profit trends.
* Understand customer purchasing behavior.
* Support data-driven business decisions.

## Tools Used

* Power BI
* Microsoft Excel / CSV Dataset
* Data Visualization
* Data Analysis

## Key Features

* Sales Overview Dashboard
* Revenue and Profit Analysis
* Product Category Performance
* Customer Insights
* Interactive Filters and Slicers
* KPI Tracking and Trend Analysis
